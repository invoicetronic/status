# Invoicetronic Status

Status page for [Invoicetronic](https://invoicetronic.com) services, powered by [Upptime](https://upptime.js.org).

Live at **[status.invoicetronic.com](https://status.invoicetronic.com)**

## Monitored services

| Service | URL |
| --- | --- |
| API | `https://api.invoicetronic.com/v1/health/` |
| Dashboard | `https://dashboard.invoicetronic.com` |
| API Reference | `https://api.invoicetronic.com/v1/docs` |

## How it works

- Uptime checks run every 5 minutes via GitHub Actions
- Response time is recorded daily
- When a service goes down, a GitHub Issue is automatically opened
- The status page is built and deployed to GitHub Pages
